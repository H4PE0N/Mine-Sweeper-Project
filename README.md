# Mine-Sweeper-Project

When you want to print out something further up in the terminal, you can use ANSI escape codes

UP:		"\033[<N>A"
DOWN:	"\033[<N>B"
RIGHT:	"\033[<N>C"
LEFT:	"\033[<N>D"

UP:		"\033[%dA", n
DOWN:	"\033[%dB", n
RIGHT:	"\033[%dC", n
LEFT:	"\033[%dD", n

n:		how many steps you want to take
